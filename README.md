# Personal Homepage

这是一个可以直接上传到 GitHub 的静态个人主页骨架。

## 当前结构

- `index.html`：首页，大图 + 左侧介绍 + 右侧四分区入口
- `styles.css`：全站共用样式
- `assets/hero-art.svg`：当前主视觉图片
- `pages/about.html`：关于我
- `pages/projects.html`：项目作品
- `pages/journey.html`：寻径页
- `pages/contact.html`：联系方式

## 适合第一轮先改的内容

1. 把 `index.html` 里的 `Your Name` 和介绍文字改成你自己的信息。
2. 如果你有自己的大图，把 `assets/hero-art.svg` 换成图片文件，并同步修改引用路径。
3. 按你自己的需要重命名四个分区和四个子页面内容。

## 上传 GitHub

1. 在 GitHub 新建一个仓库。
2. 把这个目录下的文件上传到仓库根目录。
3. 如果要用 GitHub Pages，进入仓库 `Settings > Pages`。
4. 选择部署分支，例如 `main`，目录选 `/ (root)`。
5. 保存后等待几分钟，页面就会生成。

## 为什么先用静态页面

- 没有构建工具，上传就能用。
- 后续要改成更复杂的框架时，也可以直接沿用当前内容结构。
- 适合先把网站上线，再逐步补作品、经历和联系方式。
