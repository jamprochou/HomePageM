课程主页与常用链接

  • 常用站点

    先放几个高频、稳定、适合查资料和检索问题的入口，后面再继续补充。

    arXiv Math StackExchange MathWorld NIST DLMF

  • LaTeX / 公式工具

    写公式、查符号、在线编译时最常用的一组工具，适合和课程页一起搭配用。

    Overleaf CodeCogs Detexify

  • 绘图 / CAS / 计算工具

    适合画函数图像、看几何直观、做符号计算和临时验算，数学系会高频用到。

    GeoGebra Graphing GeoGebra CAS Desmos Desmos 3D WolframAlpha

  • 画图 / 流程图 / 草稿板

    用来画思路图、课程结构图、证明草稿和整理笔记时的流程图，比纯文本更快。

    diagrams.net Excalidraw

  • 引文 / 参考文献工具

    临时生成引用格式、整理 bibliography 的时候很省事，写报告和课程论文时很好用。

    ZoteroBib

  • Notebook / 在线运行工具

    适合临时跑代码、做数值实验、写带公式的计算草稿，也适合把课程练习整理成 notebook。

    CoCalc Jupyter CoCalc LaTeX Google Colab SageMathCell

  • Markdown / 轻量写作工具

    适合快速写笔记、贴公式、临时整理推导，通常比开完整文档工程更轻便。

    edtr.io Mathpix Markdown

  • 矩阵 / 线性代数工具

    适合临时验算矩阵分解、线性方程组和基础线代步骤，不想手推时可以快速对照。

    Symbolab Matrix

  • MIT 数学基础课主页

建议整理方式

  • 每条链接都尽量附一行说明，避免过一段时间忘记它是做什么的。
  • 高频使用的站点优先放最前面，低频长文可以放到文章区。
  • 如果以后链接太多，可以继续拆成独立子页。
  • 如果某条链接对应某一门课，也可以顺手标注“线代 / PDE / 分析 / 代数 / 数论 / 复分析 / 拓扑 / 工具站”。
