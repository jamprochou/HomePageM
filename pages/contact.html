SECTION 04

联系方式

最开始不一定要放很多社交入口，但至少建议保留一个有效邮箱和 GitHub 链接。 如果你后续需要求职、合作或展示作品，这一页会是最直接的落点。

返回首页 回到关于我

第一版先保留

  • 邮箱地址。
  • GitHub 主页。
  • 一句简单说明，例如欢迎交流或合作。

以后可以补充

  • LinkedIn、Bilibili、知乎等社交入口。
  • 二维码或下载简历按钮。
  • 一个简单的留言或表单区块。

推荐写法

联系方式页越直接越好。与其写很多说明，不如把真正能联系到你的方式放在最前面， 让访问者不用找半天。
